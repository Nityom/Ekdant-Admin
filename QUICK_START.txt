=======================================================
  EKDANT DENTAL CLINIC - QUICK START GUIDE
=======================================================

Prerequisites:
  - Node.js 18+
  - npm
  - Convex deployment
  - Gmail app password (for OTP emails)

Quick Start:

1) Install dependencies
   npm install

2) Configure .env.local
   Add/verify these values:

   NEXT_PUBLIC_CONVEX_SITE_URL=https://your-deployment.convex.site
   NEXT_PUBLIC_CONVEX_URL=https://your-deployment.convex.cloud
   CONVEX_DEPLOYMENT=dev:your-deployment

  AUTH_LOGIN_EMAIL=drlalit@ekdantdentalclinics.com
   AUTH_LOGIN_PASSWORD=Admin@123
  AUTH_OTP_TO_EMAIL=drlalit@ekdantdentalclinics.com

   SMTP_HOST=smtp.gmail.com
   SMTP_PORT=587
   SMTP_SECURE=false
   SMTP_USER=your-gmail@gmail.com
   SMTP_PASS=your-16-char-gmail-app-password
   SMTP_FROM=your-gmail@gmail.com

3) Push Convex schema/functions
   npx convex dev --once

4) Start app
   npm run dev

5) Open
   http://localhost:3000

OTP Login Notes:
  - OTP is 4 digits
  - OTP validity is 12 hours
  - Resend OTP is available on login screen
  - If SMTP_USER or SMTP_PASS is missing, OTP email sending fails

Troubleshooting:
  - "Missing SMTP_USER or SMTP_PASS": set both values in .env.local and restart dev server
  - Convex errors: run `npx convex dev --once` again
  - Port in use: stop existing process on port 3000 or run on a different port

Useful Commands:
  - npm run dev
  - npm run build
  - npm run start
  - npx convex dev --once
